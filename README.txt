Página web desarrollada para una empresa constructora por Beo

Diseño moderno, limpio y responsivo con menú tipo móvil y una serie de elementos pre-diseñados. Incluye varias características técnicas interesantes en su estructura:

Está construida con componentes del proyecto Responsive Tools, desarrollado por AJ (@ajlkn).

Utiliza Flexbox, lo cual elimina la necesidad de hacks complicados y sistemas de grillas tradicionales basados en flotantes.

Incluye soporte para Sass*, con mixins y funciones útiles para organizar estilos de forma más eficiente.
(* = completamente opcional si prefieres usar CSS plano)

Código limpio, liviano y fácil de personalizar.

Pensado para adaptarse bien a dispositivos móviles, tablet y escritorio.

Créditos:

HTML5 UP (html5up.net)

Íconos: Font Awesome (fontawesome.com)

jQuery (jquery.com)

Scrollex (github.com/ajlkn/jquery.scrollex)

Responsive Tools (github.com/ajlkn/responsive-tools)
